# dotfiles

This setup is heavily inspired by [cpojer/dotfiles](https://github.com/cpojer/dotfiles) setup & his [blog post](https://cpojer.net/posts/set-up-a-new-mac-fast)

## Mac

```
git clone https://github.com/jumbosushi/dotfiles.git
cd dotfiles
./bootstrap
./homebrew
```

## Linux

```
git clone https://github.com/jumbosushi/dotfiles.git
cd dotfiles
./bootstrap
./apt
```
